# riftbound-calendar

## Installation

To install dependencies:

```bash
bun install
```

This project was created using `bun init` in bun v1.0.14. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

## Usage

Use with caution! You don't want to run this script too many times to avoid spam UVS event platform. This code is meant for once or twice in a day usage, please do not abuse it.

```bash
npm run start
```
